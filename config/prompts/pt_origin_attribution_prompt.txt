author's text is supported by a PASSAGE taken from a source they cited. Your only
job is to decide whether that passage is the source's OWN assertion, or whether

{CLAIM}

{PASSAGE}

Decide the attribution of the PASSAGE for THIS claim's content:
- "own": the passage reports the source's OWN data, experiment, finding,

- "derivative": the passage attributes the claim's content to another work —

Rules:
- Judge ATTRIBUTION only — never whether the claim is true or well-supported.
- A source describing/critiquing another study's results is derivative for those

- If the passage mixes the source's own analysis with a cited finding, classify

- If derivative, report the single reference the passage points to for this
  claim: the citation marker exactly as written ("[5]") if present, else the
  author-year or author name (e.g. "Engelmann et al. 2023").
- If you are not confident, say so with a low confidence value rather than
  guessing an attribution.

Return ONLY a JSON object, nothing else:
